# Alt+F4 定时关闭程序

不同版本的功能对比：

| 版本 | 文件类型 | 输入格式 | 特殊说明 |
|------|----------|----------|----------|
| v1.0 | Batch | 分钟数 | 简单，无管理员权限 |
| v1.2 | VBS | 分钟数 | 请求管理员权限，白名单保护系统进程 |
| v1.3 | VBS | 时:分:秒 / 分:秒 / 秒 | 更灵活的时间输入 |
| v1.4-beta | VBS | 同 v1.3 | 额外将 DownKyi.exe 加入白名单 |

## 下载

请从 [Releases](https://github.com/linchenlang/Alt-F4-Timer/) 页面下载对应版本的脚本文件。
